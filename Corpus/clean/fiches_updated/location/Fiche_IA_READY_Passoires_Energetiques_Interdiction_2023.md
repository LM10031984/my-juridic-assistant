# INTERDICTION DE LOUER LES PASSOIRES ÉNERGÉTIQUES

## CALENDRIER DES INTERDICTIONS

### ⚠️ **DEPUIS LE 1ER JANVIER 2023**
- **INTERDIT** : Location de logements dont la consommation énergétique dépasse **450 kWh/m²/an**
- Concerne les DPE classés **G+** (les pires passoires)

### ⚠️ **DEPUIS LE 1ER JANVIER 2025**
- **INTERDIT** : Location de **TOUS les logements classés G**
- Concerne environ **600 000 logements** en France

### ⚠️ **À PARTIR DU 1ER JANVIER 2028**
- **INTERDIT** : Location de **TOUS les logements classés F**

### ⚠️ **À PARTIR DU 1ER JANVIER 2034**
- **INTERDIT** : Location de **TOUS les logements classés E**
